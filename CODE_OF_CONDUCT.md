# Code of Conduct

# Our Pledge

We want the Kijani project to be a friendly, welcoming, and inclusive community for everyone interested in robotics, electronics, programming, and learning.

We are committed to making participation in this project a positive experience regardless of:

* Experience level
* Age
* Nationality
* Gender
* Appearance
* Disability
* Technical background
* Personal beliefs

Everyone should feel comfortable asking questions, sharing ideas, and learning from others.

# Expected Behaviour

Community members are encouraged to:

* Be respectful and constructive.
* Help newcomers and share knowledge.
* Assume good intentions.
* Accept feedback gracefully.
* Focus discussions on improving the project.
* Encourage experimentation and learning.
* Credit the work of others where appropriate.

# Unacceptable Behaviour

The following behaviour is not acceptable:

* Harassment, insults, or personal attacks.
* Discrimination or hateful language.
* Intimidation or bullying.
* Deliberately disruptive behaviour.
* Publishing private information without permission.
* Trolling or intentionally creating conflict.
* Spam or unrelated advertising.

# Project Philosophy

Kijani exists to make robotics and battle bots more affordable and accessible.

Beginners are welcome.

Questions are welcome.

Mistakes are expected.

Learning is encouraged.

Everyone starts somewhere.

# Reporting Problems

If you encounter behaviour that violates this Code of Conduct, please contact the project maintainer.

# Enforcement

Project maintainers reserve the right to remove comments, reject contributions, or ban participants whose behaviour harms the community or prevents productive collaboration.

These decisions are intended to protect the community and maintain a positive environment for everyone.

# Scope

This Code of Conduct applies to:

* GitHub discussions
* Issues
* Pull requests
* Documentation
* Community events
* Any spaces where the Kijani project is represented

This Code of Conduct is inspired by the Contributor Covenant and adapted for the Kijani project and its educational, open-source robotics community.
