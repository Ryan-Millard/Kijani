This folder contains the first version of the project files. The origional idea was to use bluetooth serial and have an app do the controlling for the robot.

I decided against this as the would be issues with diferent phones and we want this to work with everything.

With the idea that maybe someday someone will merge the bluetooth version with the htm,l/javascript version this code is made available.

The mian difference with the V1 and the the V2/3 boards is the enable pins for the power supplies have been removed from the newer version and the newer version has one of the power supplies removed so that we could make the board 5mm thinner. this does however make it that if your servo cannot handle the 6v  you keep it at 5v . if you are not useing the servo or it can handle the higher vboltage you can with the newer boards use that via a new enable pin.
